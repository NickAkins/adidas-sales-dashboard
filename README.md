# Adidas Sales Performance Dashboard

## Overview
This project features an  Tableau dashboard designed to analyze Adidas sales data and provide insight into current-year performance, profitability, and regional trends. The dashboard is framed to tell a story about the CY sales data and support both executive-level monitoring and detailed drill-down analysis.

## Business Questions
- How are total sales, profit, and orders performing in the current year?
- Which regions contribute most to overall sales?
- How do sales trends change over time?
- Which products and sales methods drive the strongest performance?

## Key Metrics
- Total Sales (Current Year)
- Total Profit (Current Year)
- Total Orders (Current Year)
- Profit Margin
- Profit Margin % Change (Year-over-Year)

## Dashboard Features
- KPI cards for quick performance assessment
- Sales by Region 
- Daily and monthly sales trends
- Top 3 products by sales
- Order-level detail table for transaction-level insights

## Tools Used
- Tableau Public (data visualization, forecasting, calculated fields)
- Excel (initial data review and validation)

## Key Takeaways
- Sales performance varies significantly by region
- Sales trends show seasonality and short-term fluctuations
- Profitability metrics provide deeper insight beyond total revenue
